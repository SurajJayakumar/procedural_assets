## Enhances Unreal Engine Foliage Generator By: ##

1. Smart memory allocation, management and visualization using C++ smart pointers, Unreal Engine UObject references, and low level memory tracker.
2. Smarter procedural generation taking environmental factors into account eg: Cliffs, water bodies, elevation above sea level and so on.
3. Uses a genetic algorithm for terrain generation as an alternative to true random generation
4. More customization with brush strokes, size, density, and much more!
